mpvue使用第三方vant-weapp组件库开发微信小程序注意问题

 

全局引入

在 app.json 文件中
"usingComponents": {
    "van-search": "./staticantarch/index",
    "van-sidebar": "./staticant/sidebar/index",
    "van-sidebar-item": "./staticant/sidebar-item/index",
    "van-card": "./staticant/card/index"
  }

这种模式下的小程序，在真机模拟下和开发工具中，是没有问题的，可以正常运行，但是使用开发工具的预览模式，则会报错，上传为体验版本也会报错

具体页面表现为白屏，控制台报错信息为

 

解决方案

在使用到第三方组建的文件中，在 main.json 中添加，单个添加(用到哪个，添加哪个)，没有main.json的，自己新建一个文件就好了。

main.json 

{
  "usingComponents": {
    "van-search": "../../static/vant/search/index"
    }
}

 

 

此时，问题得到了解决。

 

注意：引用文件的时候最好是把整个第三方组件库下载下来，然后放到 static 文件夹中
